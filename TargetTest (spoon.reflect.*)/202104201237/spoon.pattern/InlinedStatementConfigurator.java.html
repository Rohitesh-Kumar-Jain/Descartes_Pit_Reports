/**
36 
 * Builds inline statements of Pattern
37 
 *
38 
 * For example if the `for` statement in this pattern model
39 
 * <pre><code>
40 
 * for(Object x : $iterable$) {

 *	System.out.println(x);

 * }

 * </code></pre>
44 
 * is configured as inline statement and a Pattern is substituted
45
